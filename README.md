# Arduino Roulette (WOKWI)

Simulation of a roulette system using Arduino.

## Components

- Arduino Uno
- NeoPixel LED ring
- LCD display
- buttons (4)

## Features

- random roulette spin
- LED animation
- result displayed on LCD
